# Coin Agent — Testnet Console

Binance Spot Testnet 기반의 암호화폐 모의투자 웹 콘솔입니다.
백엔드 API 서버와 연동하여 실시간 시세 조회, 호가 확인, 캔들 차트 분석, 테스트넷 주문 생성·조회·취소 등을 수행할 수 있습니다.

> **주의**: 이 앱은 Binance **Testnet** 환경만 지원하며, 실제 자금이 사용되지 않습니다.

## 로컬 실행

### 사전 준비

- Node.js 18 이상
- 백엔드 API 서버가 `http://localhost:8000`에서 실행 중이어야 합니다.

### 설치 및 실행

```bash
# 의존성 설치
npm install

# 환경 변수 설정
cp .env.example .env

# 개발 서버 시작
npm run dev
```

### 환경 변수

| 변수 | 기본값 | 설명 |
|------|--------|------|
| `VITE_API_BASE_URL` | `http://localhost:8000` | 백엔드 API 서버 주소. 미설정 시 현재 페이지 hostname 기준 `:8000` 을 사용 |

### 스크립트

| 명령어 | 설명 |
|--------|------|
| `npm run dev` | Vite 개발 서버 실행 |
| `npm run build` | TypeScript 컴파일 + 프로덕션 빌드 |
| `npm run preview` | 빌드 결과 미리보기 |
| `npm run lint` | ESLint 코드 검사 |

## 화면 안내

### 대시보드 (`/dashboard`)

메인 화면으로, 선택한 심볼(BTCUSDT / ETHUSDT)의 시장 데이터를 한눈에 확인할 수 있습니다.

- **현재가 카드** — 선택한 심볼의 실시간 가격을 표시합니다.
- **호가 카드** — 최우선 매수·매도 호가와 상위 10단계 호가 depth를 보여줍니다.
- **캔들 차트** — 1분~1일 단위로 인터벌을 변경하며 캔들스틱 차트를 분석할 수 있습니다.
- **잔고 카드** — Testnet 계정의 자산별 가용(free) 잔액과 잠금(locked) 잔액을 확인합니다.
- **스트림 상태 카드** — 백엔드의 WebSocket 스트림 연결 상태, 마지막 이벤트의 현재가·24시간 고저·거래량 등을 표시합니다.
- **가능한 동작**: 심볼 전환, 캔들 인터벌 변경, 개별 카드 새로고침, 전체 데이터 일괄 조회.

### 주문 테스트 (`/orders`)

Testnet 환경에서 주문을 생성하고 관리하는 화면입니다.

- **주문 폼** — 심볼, 방향(매수/매도), 유형(시장가/지정가)을 선택하고 수량·가격을 입력하여 주문을 제출합니다. 시장가 매수 시에는 USDT 기준 금액(quoteOrderQty)을, 시장가 매도 시에는 수량(quantity)을 입력합니다.
- **주문 상태 조회** — 심볼과 주문 ID로 특정 주문의 현재 상태와 체결 수량을 조회합니다.
- **주문 취소** — 심볼과 주문 ID를 입력하면 확인 모달을 거쳐 미체결 주문을 취소합니다.
- **주문 로그** — 현재 세션에서 생성한 주문 내역이 시간순으로 표시됩니다. (페이지 새로고침 시 초기화)

### 에이전트 리포트 (`/reports`)

AI 에이전트의 의사결정 과정과 실행 결과를 확인하는 화면입니다. 현재 구현은 `runId` 기준 단일 리포트를 백엔드에서 실시간 조회합니다.

- **실행 리포트 카드** — `GET /api/v1/testnet/orders/report?runId=...` 응답을 기반으로 현재 run의 lifecycle, 요약, 의사결정 트레이스를 표시합니다.
- **리포트 조회 방식** — URL 쿼리 파라미터 `runId` 를 사용해 특정 실행 리포트를 직접 조회합니다.
- **케이던스/히스토리 영역** — 전용 API가 아직 없으므로 현재는 placeholder 안내를 표시합니다.

### 환경 설정 (`/settings`)

앱의 연결 설정 정보를 확인하는 화면입니다.

- **API 서버 주소** — 현재 연결된 백엔드 API URL을 표시합니다.
- **Testnet 연결 정보** — REST / WebSocket 엔드포인트 정보를 표시합니다. (일부 미구현)
- API 키는 서버 환경 변수로 관리되며, 프론트엔드에 노출되지 않습니다.
